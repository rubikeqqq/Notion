# 工业视觉选型总览

这一组笔记主要围绕工业视觉系统的硬件选型过程展开。

如果把工业视觉系统放到实际工程里理解，它和"买一个相机"或者"用 OpenCV 写一段算法"的最本质区别是：

工业视觉选型是在做一套成像系统设计，而不仅仅是挑一个设备。

也就是说：

- 选相机不是选参数最高的，而是选刚好满足精度和速度要求的；
- 选镜头不是选最贵的，而是要和传感器尺寸、工作距离、视野匹配的；
- 选光源不是选最亮的，而是要让目标特征和背景在图像上能分开的；
- 选接口不是选最新的，而是要考虑传输距离、多相机带宽和稳定性的。

## 一、这组笔记主要解决什么

如果把工业视觉选型放到真实项目里理解，这组笔记主要解决四件事：

1. 如何根据检测需求（精度、速度、视野）反推相机和镜头参数；
2. 如何选择和设计打光方案，让目标特征在图像上可区分；
3. 如何选择合适的传输接口和处理平台；
4. 如何在预算、空间、节拍等现实约束下做权衡。

## 二、可以把整组内容分成五层

### 1. 系统认知

- 工业视觉选型总览
- 与传统方案设计对比

这一层解决的是：

"工业视觉选型到底在选什么，它和普通买相机有什么不同。"

### 2. 成像核心（相机 + 镜头）

- 相机传感器
- 相机选型计算
- 镜头参数与选型
- 焦距视野计算
- 远心与特种镜头

这一层解决的是：

"要拍清楚目标，相机和镜头该怎么配。"

### 3. 光学设计（光源 + 配件）

- 光源类型
- 打光方案与配件

这一层解决的是：

"目标特征在图像上能不能被区分出来。"

### 4. 传输与计算（接口 + 控制器 + 软件）

- 相机接口标准
- 采集卡与帧抓取器
- 控制器与工业 PC
- 软件平台对比

这一层解决的是：

"图像数据怎么传、怎么算、用什么软件跑。"

### 5. 综合应用

- 完整选型案例
- 常见坑和排错清单

这一层解决的是：

"前面几层的知识怎么串起来，以及实际选型时最常踩的坑。"

## 三、如果按项目实践理解，可以这样组合

### 1. 高精度尺寸测量

- 远心镜头
- 高分辨率相机
- 背光源
- 标定与像素当量

例如：精密零件尺寸检测、手机中框尺寸测量。

### 2. 表面缺陷检测

- 面阵相机
- 合适的打光方案（明场/暗场根据缺陷类型定）
- 偏振或滤光处理反光

例如：金属表面划痕、PCB 焊点缺陷、薄膜针孔。

### 3. 定位 + 读码识别

- 合适分辨率的相机
- 低畸变镜头
- 均匀漫射光源或同轴光

例如：产品定位后读取二维码或序列号。

### 4. 高速在线检测

- 高帧率相机
- 合适的接口（USB3 / CXP）
- 采集卡或高性能 PC

例如：产线全检、飞拍应用。

## 总结

1. 工业视觉选型的本质是在约束条件下做权衡，不是参数越高越好；
2. 这组内容按"成像 → 光学 → 传输 → 计算 → 案例"五层组织，覆盖完整选型流程；
3. 如果只学理论计算而不了解光源和打光的实际影响，项目里很容易写出"精度算够了但实际效果不行"的方案；
4. 这组笔记的目标不是记住所有参数规格，而是能在实际项目中独立完成一套视觉系统的选型设计。
