# 打光方案设计与光学配件选型

前一篇介绍了光源的类型和基本选型思路，这一篇更进一步：针对具体检测任务，如何设计打光方案，以及如何搭配滤光片、偏振片、漫射器等光学配件。

## 一、明场与暗场

这是打光方案中最基本也最重要的两个方向。

### 明场照明

光线反射进入相机镜头，目标在图像中显亮，背景显暗。

适合：

- 表面光滑的目标；
- 需要增强目标本身的亮度；
- 字符识别、印刷检测。

### 暗场照明

光线不直接进入相机镜头，只有目标表面的凸起、划痕、边缘等结构使光线散射进入镜头。

适合：

- 低角度环形光或条光，只照亮凸起或凹陷的边缘；
- 表面划痕、凹凸、浮雕检测。

### 明场 vs 暗场的选择

同一个目标特征，用明场和暗场拍出来的图像对比度完全不一样。

例如一个刻印字符：

- 明场下字符可能显暗，背景显亮；
- 暗场下字符边缘可能显亮，背景显暗。

选哪种看哪种能让检测特征在图像中更明显。

## 二、不同材料的打光策略

### 金属表面

金属是高反光材料，常见问题是大面积反光导致图像局部过曝。

策略：

- 使用漫射光源（圆顶光）或同轴光；
- 搭配偏振片使用，滤除镜面反射分量；
- 避免单一方向强光直射。

### 玻璃/透明材料

透明材料的检测挑战在于它既透光又反光。

策略：

- 背光照明做轮廓测量效果最好；
- 表面缺陷用暗场照明（划痕在暗场下显亮）；
- 需要看清内部结构时选择透射照明。

### 塑料

塑料表面的打光取决于塑料的颜色和表面质地：

- 透明塑料：类似玻璃方案，背光或暗场；
- 白色/浅色塑料：环形光或同轴光均可，注意控制反光；
- 黑色塑料：需要大光圈或强光源补偿曝光量。

### PCB/电子元件

PCB 表面有焊点、走线、标记等多种特征。

策略：

- 环形光适合焊点检测（环形光让焊点呈现立体感）；
- 同轴光适合平面走线和标记；
- 多角度组合照明可以同时展示多种特征。

## 三、滤光片的选型

滤光片的核心作用是选择性地透射或阻挡特定波长的光。

### 带通滤光片

只允许某个特定波长范围内的光通过。

用途：

- 和环境光隔离——配合单色光源使用，过滤掉环境光中其他波长的干扰；
- 增强对比度——只允许目标的特征波长的光通过。

### 长通/短通滤光片

- 长通滤光片：通过波长大于某个阈值的光；
- 短通滤光片：通过波长小于某个阈值的光。

用途：

- 红外滤光片（长通，透过可见光阻挡红外）：普通场景减少红外干扰；
- 紫外滤光片（短通）：紫外荧光检测场景。

### 偏振滤光片

偏振片的工作原理在前一篇已经提过。

在实际打光中，偏振片的使用有两种常见模式：

- 光源偏振 + 镜头同向偏振：抑制环境光，但保留目标反射的偏振分量；
- 光源偏振 + 镜头正交偏振：消除镜面反射，只保留漫反射。

在金属和玻璃检测中，正交偏振方案非常有效。

## 四、漫射器

漫射器的作用是把方向性强的光线变成均匀散射光。

用途：

- 减少点光源造成的热点和阴影；
- 获得更均匀的照明。

在环形光或条形光上加装漫射板，可以显著改善高反光表面的成像均匀性。

代价：光线会有一定损失，亮度会下降。

## 五、多角度组合照明

在很多实际检测场景中，单一光源很难同时满足多个检测需求。

常见的组合方案：

- 同轴光 + 环形光：同轴光提供均匀平面照明，环形光补充边缘和立体感；
- 背光 + 正面光：背光测轮廓，正面光看表面细节，不同采集任务切换使用；
- 多角度条光：从不同方向逐次采集图像，组合分析（常用于高反光曲面检测）。

组合照明增加了硬件成本和采集时间，但可以显著提升检测效果。

## 六、安装与机械考虑

### 1. 工作距离

光源的工作距离（光源到目标的距离）影响光照强度和均匀性：

- 距离太近：光照集中但可能不均匀，容易产生热点；
- 距离太远：光照均匀但强度下降。

需要在均匀性和亮度之间做权衡。

### 2. 角度

光源的照射角度直接影响图像对比度：

- 高角度照明（接近垂直）：减少阴影，适合平整表面；
- 低角度照明（接近水平）：增强边缘和凹凸感；
- 可调角度的光源支架在打光调试阶段非常有用。

### 3. 光源散热

长时间工作的光源会产生热量：

- 热量会影响光源寿命（LED 光衰和温度强相关）；
- 高亮频闪模式比常亮模式的热负担更低；
- 安装空间要考虑散热。

### 4. 光源控制器

选择光源时需要考虑配套的控制器：

- 亮度是否可调；
- 是否支持频闪模式；
- 是否支持相机触发同步。

## 七、打光方案验证流程

一个推荐的打光验证流程：

1. 了解目标特性和检测需求；
2. 根据目标材质和特征类型选择 2-3 个候选光源方案；
3. 用实际样品（OK 样和 NG 样）打光测试；
4. 评估图像对比度和均匀性；
5. 在不同光照条件下验证稳定性；
6. 确认方案后再固化安装方式。

绝大部分视觉项目，打光实验阶段花的时间越充分，后续算法调试阶段越顺利。

## 八、光学配件选型检查清单

1. 滤光片 → 是否需要隔离环境光？是否要增强特定波长对比度？
2. 偏振片 → 是否有反光干扰需要消除？
3. 漫射器 → 是否需要更均匀的光照？
4. 光源支架 → 是否支持角度和位置调节？
5. 光源控制器 → 是否支持亮度和频闪控制？
6. 散热 → 长时间工作是否满足散热要求？

## 总结

1. 打光方案的选择（明场/暗场/组合）直接影响图像对比度，是检测系统设计的关键环节；
2. 不同材料有各自最佳的打光策略，没有通用的"万能方案"；
3. 偏振和滤光是处理反光和抑制环境光的有效手段；
4. 打光实验验证是落地方案前不可或缺的步骤，不要跳过；
5. 光学配件的选型（滤光片、偏振片、漫射器、控制器）和光源本身一样重要。
