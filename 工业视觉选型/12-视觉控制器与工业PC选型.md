# 视觉控制器与工业 PC 选型

相机、镜头、光源和接口都选好了，下一步是：图像数据传到什么设备上处理。

视觉控制器或工业 PC 的选型，很多人会觉得"不就是一台电脑吗"，但实际上工业视觉场景对工控机的要求和普通办公电脑有很大区别。

## 一、工业 PC 和普通电脑的区别

### 稳定性与可靠性

工业 PC 的设计目标是"长时间运行不关机、在恶劣环境下稳定工作"：

- 宽温设计（通常 -20°C 到 60°C 或更高）；
- 防尘防振（无风扇设计、固态硬盘、加固连接器）；
- 长生命周期（同一型号供应 5 年以上，方便项目复制）；
- 工业级元器件（更宽的温度和电压容忍范围）。

普通商用 PC 在这些方面都不满足工业现场的要求。

### I/O 接口

工业 PC 通常需要额外的接口用于连接外围设备：

- 数字 I/O（触发传感器、频闪灯、PLC 通信）；
- 串口（RS232/RS485，用于和自动化设备通信）；
- 多个网口（多相机 GigE 或多网段隔离）；
- PCIe 插槽（插采集卡或运动控制卡）。

## 二、三类常见的视觉处理平台

### 1. 无风扇嵌入式工控机

特点：

- 体积小，可安装在设备内部或控制柜中；
- 无风扇设计，无尘、低噪、高可靠性；
- CPU 通常为低功耗型号（Intel Core i5/i7 U 系列或 N 系列）。

适用场景：

- 设备集成度高、空间受限；
- 检测任务计算负载不是特别大；
- 环境有粉尘或油雾（无风扇避免吸入污染物）。

局限：

- 扩展性有限（PCIe 插槽少或无）；
- CPU 性能不如同价位的台式工控机；
- 散热受限，不能长时间高负载运行。

### 2. 工业级台式/机架式工控机

特点：

- 性能强（可选桌面级 CPU 甚至工作站 CPU）；
- 扩展性好（多 PCIe 插槽、多硬盘位、多 I/O 接口）；
- 散热好，可以长时间满负荷运行。

适用场景：

- 多相机系统（需要多个采集卡）；
- 算法复杂、计算量大（如深度学习检测）；
- 需要大量图像存储和日志记录。

局限：

- 体积大，占用空间；
- 价格通常高于嵌入式方案；
- 带风扇，需要定期清理。

### 3. 专用视觉控制器

一些相机品牌（如 Cognex、Keyence）提供集成了视觉软件的专用控制器。

特点：

- 软硬件一体，开箱即用；
- 稳定性高，不需要自己配置驱动和环境；
- 通常配有专用 I/O 和通信接口。

适用场景：

- 使用品牌自家相机和软件；
- 项目标准化程度高；
- 对集成复杂度敏感（希望尽可能简化）。

局限：

- 价格通常高于组装的工控机方案；
- 只能在品牌的生态内使用。

## 三、CPU 选型

### 核心数 vs 频率

工业视觉软件（Halcon、VisionPro）的性能通常与 CPU 单核性能和核心数量都有关系。

- 算法计算阶段：更依赖单核性能（算子执行通常是单线程或有限多线程）；
- 多检测流程并行：需要更多核心；
- 深度学习推理：需要 GPU 或高核心数 CPU。

### 常见选型经验

- 简单检测（1-2 个定位/测量项）：i5 或同级别即可；
- 中等复杂度检测（多个检测项同时执行）：i7 或 Xeon E 系列；
- 复杂检测（含深度学习或并行多检测）：i9 / Xeon W 或以上。

## 四、GPU 选型

是否需要独立 GPU，取决于是否使用深度学习：

### 需要 GPU 的场景

- 深度学习缺陷检测；
- 需要 GPU 加速的 Halcon 算子（某些匹配、分类算子支持 GPU）；
- 大量图像预处理运算。

### 不需要 GPU 的场景

- 传统视觉算法（定位、测量、Blob、OCR）；
- 单相机低分辨率场景；
- 处理速度已经满足节拍的场景。

GPU 选型的建议：

- 深度学习场景：NVIDIA RTX 系列（显存 > 8GB）；
- 传统算法加速：普通 GPU 或核芯显卡即可；
- 注意工控机电源和散热是否支持独立 GPU 的功耗。

## 五、内存与存储

### 内存

工业视觉系统对内存的需求通常取决于：

- 图像大小（分辨率 × 像素位深）；
- 缓冲帧数（同时保留多少帧在内存中）；
- 操作系统和软件占用。

常见配置：

- 常规检测：16GB 够用；
- 多相机高分辨率：32GB 或 64GB。

### 存储

- 系统盘：SSD（建议 M.2 NVMe），256GB-512GB；
- 数据盘：保存检测图像和日志，按需求配置大小；
- 注意：工业 PC 应使用工业级 SSD（宽温、高寿命）。

## 六、数字 I/O 与设备通信

工业视觉系统通常需要和 PLC、传感器、执行机构通信。

选型时需要考虑的 I/O 需求：

- 触发输入：传感器信号接入，触发相机采集；
- 闪光灯输出：同步触发频闪光源；
- 结果输出：OK/NG 信号发给 PLC；
- 串口通信：RS232/RS485 与 PLC 或条码枪通信；
- 以太网通信：TCP/IP、Modbus TCP、Profinet 等。

选工业 PC 时确认 I/O 点数是否满足需求，或者是否预留了扩展接口。

## 七、环境与防护

工业 PC 需要满足安装环境的防护要求：

- 工作温度：确认现场是否有空调 / 是否在高温区域；
- IP 防护等级：是否有粉尘或液体飞溅；
- 抗振性：是否有振动源（安装在机器本体上时）；
- EMC 电磁兼容：是否有强电磁干扰。

选型时查看工控机的产品规格书，确认环境参数匹配。

## 八、视觉控制器选型检查清单

1. 平台类型 → 嵌入式 / 台式工控机 / 专用控制器；
2. CPU → 是否满足算法计算需求；
3. GPU → 是否需要深度学习加速；
4. 内存 → 是否满足多帧缓存需求；
5. 存储 → SSD 类型和容量是否足够；
6. I/O → 数字 I/O、串口、网口数量是否满足设备通信需求；
7. 扩展性 → PCIe 插槽是否满足采集卡数量需求；
8. 环境 → 温度、防护、抗振是否匹配现场条件；
9. 品牌 → 优先选和视觉软件有合作认证的品牌（如 Advantech、ADLINK）。

## 总结

1. 工业 PC 不是普通电脑——它的稳定性和接口扩展性是工业场景的关键要求；
2. 嵌入式工控机适合空间受限和低负载场景，台式工控机适合多相机和高性能场景；
3. GPU 只有在使用深度学习时才必要，传统视觉场景对 GPU 依赖很低；
4. I/O 和通信接口的配置在选型时容易被忽略，但这部分是项目集成的关键；
5. 选型时注意环境参数（温度、防护、抗振），否则工控机在现场可能频繁故障。
