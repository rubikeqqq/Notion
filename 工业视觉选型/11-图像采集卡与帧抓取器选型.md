# 图像采集卡与帧抓取器选型

"我到底需不需要图像采集卡？"这是在工业视觉选型中经常被问到的问题。

答案取决于相机接口类型和系统配置。

这篇把采集卡和帧抓取器的选型场景讲清楚。

## 一、采集卡并不是所有场景都需要的

先区分两个容易混淆的概念：

### 普通网络接口卡（NIC）

对于 GigE Vision 相机，使用标准的千兆以太网卡就可以工作，不需要专用的图像采集卡。

大多数工控机主板的板载网卡就能满足常规 GigE 相机的需求。

### 专用图像采集卡（Frame Grabber）

专用采集卡（也叫帧抓取器）在以下几种场景下是必需的或强烈推荐的：

- Camera Link 接口相机（必须使用采集卡，不能直接连接电脑）；
- CoaXPress 接口相机（需要 CXP 主机适配器，也是一种采集卡）；
- 高负载 GigE 多相机系统（板载网卡可能不够稳定）；
- 需要精确多相机硬件同步的场景。

## 二、哪些场景需要专用采集卡

### 1. Camera Link 相机

Camera Link 接口没有对应的电脑原生接口标准，必须通过 Camera Link 采集卡才能把相机连接到计算机。

选采样集卡时需要注意：

- Base/Medium/Full 配置的通道数；
- 板载内存大小（决定突发传输能力）；
- PCIe 接口速度（确保不成为瓶颈）。

### 2. CoaXPress 相机

CoaXPress 也需要专用的主机适配器（CXP 采集卡）。

选型注意：

- CXP 链路数（1 根线、2 根线还是 4 根线）；
- CXP 版本（CXP-6、CXP-12）；
- 板载内存；
- PCIe 通道数和速度。

CXP 采集卡通常会标注支持的相机数量，例如"4 通道 CXP-12 采集卡"可以接 4 个单链路 CXP-12 相机。

### 3. 高负载 GigE 多相机系统

如果系统中有多个 GigE 相机同时高帧率运行，板载网卡可能会出现：

- 带宽争抢（多个相机共享同一网卡带宽）；
- 中断处理延迟导致丢帧；
- 时间戳同步精度不足。

此时可以用以下几类方案提升稳定性：

- 多网卡（每个相机独占一张网卡）；
- 专用 GigE 采集卡（专门为图像传输优化，减少 CPU 负载）；
- 支持 IEEE 1588（精确时间协议）的网卡。

### 4. 需要精确硬件同步的多相机系统

如果多个相机需要在同一时刻精确同步采集（例如 3D 重建、多视角检测），采集卡的作用就很关键。

采集卡可以提供：

- 硬件触发信号分发；
- 精确的相机曝光同步；
- 时间戳标记。

用采集卡的硬件触发输出，比软件管理触发信号的同步精度高一个数量级。

## 三、板载内存的作用

采集卡的板载内存大小是一个容易被忽略但实际影响很大的参数。

- 小内存采集卡：传输负担重，容易丢帧；
- 大内存采集卡：可以缓存多帧图像，允许系统突发传输。

在高速相机场景中，大板载内存的采集卡可以显著降低丢帧风险。

常见配置：512MB、1GB、2GB 采集卡板载内存。

## 四、PCIe 接口匹配

采集卡通过 PCIe 连接到工控机主板性能受限于 PCIe 通道数和版本。

带宽需求示例：

- Camera Link Full 配置：约 5.4 Gbps，需要至少 PCIe 2.0 ×4；
- CoaXPress CXP-12 ×4：约 50 Gbps，需要 PCIe 3.0 ×8 或更高。

选型时确保：

1. 工控机有足够的物理 PCIe 插槽；
2. 插槽的 PCIe 版本和通道数满足采集卡带宽需求；
3. BIOS 配置没有限制 PCIe 带宽。

## 五、多相机同步方案对比

### 软件同步

- 依赖网络或 USB 协议发送触发指令；
- 同步精度通常在毫秒级；
- 成本最低，不需要额外硬件。

### 硬件触发线

- 通过相机 I/O 接口直接发送硬件触发信号；
- 同步精度在微秒级；
- 需要额外布线，需要光源控制器支持。

### 采集卡硬件同步

- 采集卡统一分发触发信号给多个相机；
- 同步精度在亚微秒级；
- 需要采集卡支持和专用线缆。

## 六、采集卡选型检查清单

1. 相机接口类型 → Camera Link / CXP 需要采集卡，GigE 根据场景可选；
2. 采集卡通道数 → 是否满足相机数量和带宽需求；
3. 板载内存 → 是否满足缓存和防丢帧需求；
4. PCIe 接口 → 是否匹配工控机插槽版本和通道数；
5. 触发同步 → 是否需要采集卡提供硬件同步；
6. 软件兼容性 → 采集卡的 SDK 是否兼容视觉软件（Halcon / VisionPro）；
7. 品牌和售后 → 工业项目的采集卡尽量选主流品牌（Euresys、BitFlow、Active Silicon 等）。

## 七、主流采集卡品牌

几个常见的工业采集卡品牌：

- Euresys：产品线最全，CoaXPress 和 Camera Link 都有覆盖，软件兼容性好；
- BitFlow：Camera Link 采集卡老牌厂商，稳定性好；
- Active Silicon：以 FireBird 系列闻名，支持多种接口；
- National Instruments（NI）：视觉采集卡和控制器方案，适合集成测试系统。

选型时优先考虑项目视觉软件（Halcon / VisionPro）已经提供了原生支持的采集卡品牌，这样可以节省大量驱动和集成调试时间。

## 总结

1. 不是所有项目都需要采集卡——GigE 相机用板载网卡就够的场景很多；
2. Camera Link 和 CoaXPress 相机必须配套使用采集卡，不能直接连接电脑；
3. 多相机高速同步系统强烈建议使用采集卡，软件同步的精度和稳定性不够；
4. 采集卡选型需考虑板载内存、PCIe 通道数、通道带宽和软件兼容性；
5. 选采集卡时优先考虑和视觉软件有原生支持的品牌，减少集成风险。
