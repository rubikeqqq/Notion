# 光源类型与选型思路

在工业视觉项目中有一个广泛的共识：

光源选型和打光方案设计，是决定检测系统成败最关键的因素之一。

一个检测项目，光源打好了，用普通相机也能稳定出结果；光源没打好，用再好的相机和镜头也救不回来。

这篇先把光源的类型和基础选型思路梳理清楚。

## 一、为什么光源这么重要

工业视觉中光源的根本目的不是"照亮"，而是"增强对比度"。

具体来说：

- 让目标特征（缺陷、边缘、字符）和背景在图像上能被明显区分开；
- 抑制不必要的细节（反光、纹理、无关结构）；
- 降低环境光的干扰，让系统在不同光照条件下稳定工作。

对比度比亮度更重要，是光源选型的第一原则。

## 二、光源几何类型

### 1. 环形光源

环形光源是最常见的光源类型，灯珠排列成环形，从目标上方倾斜照射。

特点：

- 安装方便，中间镂空不遮挡视野；
- 提供倾斜照明，可以突出边缘和凹凸特征；
- 适合大多数通用检测场景。

适用场景：

- 字符识别；
- 边缘检测；
- 通用外观检查。

### 2. 条形光源

条形光源是 LED 排列成条形，从一个或两个方向照射。

特点：

- 方向性强，可以突出特定方向的缺陷；
- 常成对使用（对射或双侧照明）。

适用场景：

- 大尺寸产品检测；
- 划痕检测（条光方向垂直于划痕方向效果更好）；
- 印刷检测。

### 3. 背光源

背光源从目标背后照射，相机和光源在目标两侧。

特点：

- 获得高对比度的轮廓图像（目标呈黑色剪影，背景为亮色）；
- 完全消除表面细节，只显示外形轮廓；
- 适合尺寸测量。

适用场景：

- 尺寸测量；
- 外形轮廓检测；
- 毛刺检测；
- 通孔/缺口检测。

### 4. 同轴光源

同轴光通过分光镜，让光线沿镜头光轴方向垂直照射目标。

特点：

- 光线垂直照射目标表面，反射光沿原路返回镜头；
- 适合高反光表面（镜面、金属、晶圆）；
- 可以消除阴影，获得均匀照明。

适用场景：

- 镜面表面检测；
- 晶圆划痕检测；
- 高反光金属字符识别。

### 5. 圆顶光源（漫射光源）

圆顶光源是半球形结构，LED 光线经过半球内壁漫反射后均匀照射目标。

特点：

- 光线从各个角度均匀照射目标，几乎没有方向性；
- 可以消除阴影和反光；
- 成像柔和，对比度偏软。

适用场景：

- 高反光曲面物体；
- 球面、弧面零件；
- 需要消除任何阴影和眩光的场景。

### 6. 线光源

线光源为线扫描相机配套设计，光线聚焦成一条细线。

特点：

- 光照集中，亮度高；
- 配合线扫描相机使用。

适用场景：

- 线扫描系统；
- 连续卷材检测；
- 大幅面高精度扫描。

## 三、光源颜色选择

### 1. 基本原理

不同颜色的光在目标表面的反射率和吸收率不同：

- 某种颜色的光照射同色物体时，反射率高（图像中显亮）；
- 某种颜色的光照射补色物体时，吸收率高（图像中显暗）。

### 2. 常用颜色及其典型应用

| 颜色 | 波长 | 典型应用 |
|------|------|----------|
| 白色 | 全光谱 | 通用，适用于大多数场景 |
| 红色 | 620-750nm | 穿透性强，适合透射检测 |
| 蓝色 | 450-495nm | 短波长，分辨细节能力强 |
| 绿色 | 495-570nm | 增强特定色彩对比度 |
| 红外 | 850-940nm | 穿透不透明材料，环境光抑制 |
| 紫外 | 365-395nm | 荧光激发，特殊材料检测 |

### 3. 颜色选型策略

- 需要增强目标特征 → 使用和目标特征对比度最大的颜色；
- 需要抑制背景 → 使用和背景颜色相近的光（背景显亮），或吸收强的光（背景显暗）；
- 环境光干扰大 → 使用和环境光波长差异大的颜色，配合滤光片更佳。

## 四、频闪 vs 常亮

### 频闪照明

- 在相机曝光瞬间以极高亮度点亮光源（通常宽脉冲驱动）；
- 可以在极短曝光时间内提供充足光照，冻结运动；
- 降低光源发热，延长寿命。

### 常亮照明

- 光源持续点亮；
- 适合低速检测或静止目标检测；
- 亮度相对频闪较低。

大多数产线检测项目选频闪照明——既能冻结运动，又能降低光源热衰减。

## 五、光源附件：偏振片

光源配合偏振片使用，可以有效抑制反光和眩光：

- 在光源前加偏振片，使照射光偏振化；
- 在镜头前加同向偏振片，可以滤除镜面反射；
- 在镜头前加正交偏振片，可以只保留漫反射光，完全消除镜面反射。

偏振方案在高反光表面的检测中非常常用，如金属表面、玻璃、薄膜。

## 六、光源选型的一般流程

1. 分析目标特征：
   - 表面是漫反射还是镜面反射？
   - 要检测的特征是凸起、凹陷、字符还是轮廓？
   - 背景是否有干扰？

2. 选择光源几何类型：
   - 轮廓测量 → 背光；
   - 表面缺陷 → 条光、环形光；
   - 反光表面 → 同轴光或圆顶光。

3. 选择颜色：
   - 根据目标颜色和背景颜色的反射特性选择。

4. 确定工作模式：
   - 运动场景 → 频闪；
   - 静止场景 → 常亮或频闪均可。

5. 做打光实验验证（这是最重要的一步）：
   - 用实际样品测试不同光源、不同角度；
   - 评估对比度是否足够。

## 七、常见光源类型应用总结

| 光源类型 | 最佳场景 | 不宜场景 |
|----------|----------|----------|
| 环形光 | 通用、字符、边缘 | 高反光平面 |
| 条形光 | 大视野、划痕 | 需要均匀照明的场景 |
| 背光 | 尺寸测量、轮廓 | 需要表面细节 |
| 同轴光 | 高反光表面 | 粗糙表面（无明显优势） |
| 圆顶光 | 曲面、反光物体 | 需要高对比度边缘 |
| 线光源 | 线扫描系统 | 面阵相机通用场景 |

## 总结

1. 光源的首要目标不是照亮，而是增强目标特征和背景之间的对比度；
2. 光源选型是几何类型、颜色、照明方式三个维度的组合选择；
3. 没有通用的"最佳光源"，每类光源都有最适合的场景和不适合的场景；
4. 打光方案必须用实际样品做实验验证，光靠理论分析可能和实际效果差异很大；
5. 光源往往是整个视觉系统里成本最低但影响最大的部件，值得花时间选好。
