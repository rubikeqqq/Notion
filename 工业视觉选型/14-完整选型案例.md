# 完整选型案例

前面把工业视觉选型的各个维度分篇讲完了。

这一篇把这些知识串起来，从一个具体的检测项目需求出发，一步步完成从需求分析到最终 BOM 的完整选型过程。

## 一、项目场景描述

### 检测对象

手机中框尺寸检测。

![](https://via.placeholder.com/300x200?text=手机中框示意图)

### 检测需求

| 项目 | 参数 |
|------|------|
| 最大产品尺寸 | 160mm × 80mm |
| 检测视野 | 170mm × 90mm（含产品放置公差） |
| 检测精度 | ±0.02mm |
| 检测节拍 | 5 秒/件（含采图和处理） |
| 检测项目 | 外形长宽、四角圆弧半径、摄像头孔位置 |
| 目标材质 | 铝合金（喷砂 + 阳极氧化处理） |
| 环境 | 产线有振动，无空调，室温可达 40°C |
| 工作距离 | 不超过 400mm（设备内部安装空间限制） |

## 二、相机选型

### 第一步：精度 → 分辨率

精度要求：±0.02mm，按特征覆盖 4 个像素的要求：

像素当量 ≤ 0.02mm × 4 = 0.08mm/pixel

更好的余量设计，像素当量取精度要求的一半：

像素当量（目标）≈ 0.01mm/pixel

视野 170mm × 90mm：

宽方向需求：170 ÷ 0.01 = 17000 pixel
高方向需求：90 ÷ 0.01 = 9000 pixel

这个分辨率太高了（约 1.5 亿像素），没有这样的工业相机。

### 第二步：调整视野和精度

实际上很难用一个相机覆盖 170mm 视野同时达到 ±0.02mm 精度。

常规做法是分两个方向考虑：

方案 A：降低视野要求 — 分两个工位测量（先测左边再测右边）
方案 B：接受更低的精度 — 如果 ±0.05mm 可以接受
方案 C：保持精度 — 用更高分辨率相机

按方案 A 分两个工位，每个工位视野 85mm × 90mm：

宽方向需求：85 ÷ 0.01 = 8500 pixel
高方向需求：90 ÷ 0.01 = 9000 pixel

仍然很高，达到 7650 万像素。

### 第三步：实际选型折中

工业相机实际可供选择的高分辨率范围：

- 1200 万像素（4096 × 3072）—— 像素当量：170÷4096 ≈ 0.042mm/pixel
- 2000 万像素（5472 × 3648）—— 像素当量：170÷5472 ≈ 0.031mm/pixel
- 2500 万像素（5120 × 5120）—— 像素当量：170÷5120 ≈ 0.033mm/pixel
- 6500 万像素（9344 × 7000）—— 像素当量：170÷9344 ≈ 0.018mm/pixel

考虑 6500 万像素相机：

像素当量 0.018mm/pixel

特征覆盖 3-4 像素时理论可分辨：0.054-0.072mm

这个精度不够 ±0.02mm。

### 第四步：重新审视方案

当相机参数无法一次满足所有需求时，需要调整测量方案。

最终可行方案：

- 使用 2000 万像素相机（5472 × 3648）；
- 像素当量：170 ÷ 5472 ≈ 0.031mm/pixel；
- 特征覆盖 3 像素时理论可分辨 0.093mm，达不到 ±0.02mm 要求。

所以此处需要引入远心镜头 + 多工位的思路，或者接受几个方案：

**最终方案**：采用 2 个工位 + 远心镜头方案

- 分 2 个工位，每个视野 90mm × 90mm；
- 每个工位用 2500 万像素（5120 × 5120）相机 + 低畸变远心镜头；
- 像素当量：90 ÷ 5120 ≈ 0.0176mm/pixel；
- 配合远心镜头消除透视误差；
- 特征覆盖 3-4 像素，可分辨约 0.053-0.07mm。

再加亚像素定位（亚像素精度可达 0.1-0.5 像素），实际测量精度可达到：

0.0176 × 0.3 ≈ 0.0053mm（亚像素 0.3 像素）

所以 ±0.02mm 精度是可行的。

**相机规格**：

- 分辨率：5120 × 5120（2500 万像素）
- 传感器尺寸：约 1.1 英寸
- 像元尺寸：约 4.5μm
- 全局快门
- 黑白
- 帧率：≥ 30fps（足够）

## 三、镜头选型

### 远心镜头参数计算

视野：90mm × 90mm
传感器尺寸：约 1.1 英寸（宽约 14mm）

放大倍率 M = 14 ÷ 90 ≈ 0.156

需要选型的远心镜头参数：

- 视野（最大可测物体）：≥ 90mm
- 放大倍率：0.156×
- 工作距离：根据镜头规格确定（通常 100-200mm）
- 远心度：≤ 0.05°
- 畸变：≤ 0.02%
- 分辨率：匹配 4.5μm 像元（≥ 110 lp/mm）

### 接口

相机和镜头都选 C 口（1.1 英寸传感器需要镜头像面尺寸覆盖）。

## 四、光源选型

### 目标分析

手机中框是铝合金喷砂 + 阳极氧化表面，有微弱反光但不是镜面。

测量需求是外形轮廓，不是表面细节。

### 光源方案

**背光源**是最佳选择：

- 背光照明获得高对比度轮廓图像（产品黑色剪影，背景亮白）；
- 完全适合尺寸测量场景；
- 配合远心镜头效果更好。

光源规格：

- 类型：LED 背光源
- 发光面积：≥ 100mm × 100mm
- 颜色：红色（穿透性好，减少环境光干扰）
- 模式：频闪控制

## 五、接口选型

数据传输量：

- 分辨率：5120 × 5120 × 8bit = 约 25 MB/帧
- 帧率：30fps
- 每秒数据量：25 × 30 = 750 MB/s ≈ 6 Gbps

带宽要求 6 Gbps：

- GigE Vision（1 Gbps）→ 不够
- USB3 Vision（5 Gbps）→ 接近但偏紧（有效约 3.5-4 Gbps）
- CoaXPress CXP-6 ×2（12.5 Gbps）→ 足够

最终接口选择：**CoaXPress CXP-6 ×2**

需要 CXP 采集卡，2 个相机各用 2 根 CXP-6 同轴线缆。

## 六、采集卡选型

2 个 CoaXPress 相机，每个相机 2 根 CXP-6 线缆。

采集卡规格：

- 支持 CXP-6 标准
- 4 通道，覆盖 2 个相机各 2 链路
- 板载内存 ≥ 1GB
- PCIe 3.0 ×8

## 七、控制器选型

### 计算负载分析

- 2 个 2500 万像素相机采集；
- 亚像素定位 + 尺寸测量 + 圆弧拟合 + 孔位置测量；
- 节拍 5 秒。

### 控制器规格

- Intel Core i7（12 代以上），或 Xeon W 系列
- 32GB DDR4 或以上
- 512GB NVMe SSD + 1TB 数据盘
- 支持 CXP 采集卡的 PCIe 插槽
- 数字 I/O：触发输入 + OK/NG 输出
- 工作温度：0-50°C
- 无风扇或带工业级风扇
- 品牌：Advantech 或 ADLINK 工控机

## 八、软件平台选型

选择依据：

- 需要高精度亚像素测量；
- 需要远心镜头标定支持；
- 需要强大的测量和几何拟合工具；
- 需要和 CXP 相机的驱动集成。

**推荐：Halcon**

- 使用 Halcon 的标定工具做像素当量和畸变校正；
- 使用测量工具做边缘检测和尺寸测量；
- 使用亚像素边缘提取达到 ±0.02mm 精度。

## 十、方案验证建议

采购前建议先做以下验证：

1. 用实际手机中框样品配合远心镜头 + 背光源打光测试，确认图像对比度满足要求；
2. 用 Halcon 标定板做标定，确认像素当量和畸变校正后的测量精度能达到 ±0.02mm；
3. 用 CXP 相机 + 采集卡验证传输稳定性和帧率；
4. 用 50 个以上 OK 和 NG 样品验证测量稳定性和重复性。

## 总结

1. 选型是一个反复迭代的过程——第一次计算可能发现无法同时满足所有需求，需要调整方案或增加工位；
2. 高精度 + 大视野的组合往往需要分拆工位或使用远心镜头 + 亚像素技术；
3. 每个选型环节（相机 → 镜头 → 光源 → 接口 → 采集卡 → 控制器 → 软件）都依赖于前面的决策；
4. 选型结束不代表设计完成，样机验证是确认选型正确性不可或缺的步骤。
