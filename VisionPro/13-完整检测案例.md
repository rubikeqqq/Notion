# 完整检测案例

前面几篇分别介绍了 VisionPro 的各项核心能力。

这一篇把前面学到的所有知识串起来，做一个完整的检测案例。

## 案例场景：手机中框检测

假设我们要检测一个手机中框，需要完成以下检测项：

1. **定位中框** —— 找到中框在产品中的位置和角度；
2. **测量孔径** —— 检测中框上的某个安装孔直径；
3. **检测表面缺陷** —— 检查中框表面有无划痕或磕碰；
4. **读取 DPM 码** —— 读取中框上的激光刻印码。

## 一、工具链设计

```

输入图像
    │
    ▼
┌─────────────┐
│ CogPMAlign  │  定位中框
└──────┬──────┘
       │ 定位结果（位置+角度）
       ▼
┌─────────────┐
│ CogFixture  │  坐标对齐
└──────┬──────┘
       │ 摆正后的图像
       ├───────────────────────┐
       ▼                       ▼
┌──────────────┐    ┌──────────────┐
│ CogFindCircle│    │  CogBlobTool │  测量 + 缺陷
└──────┬───────┘    └──────┬───────┘
       │                    │
       ▼                    ▼
┌──────────────┐    ┌──────────────┐
│  CogIDTool   │    │  结果汇总    │  识别 + 判定
└──────────────┘    └──────────────┘
       │
       ▼
   输出结果
```

## 二、代码实现

### 1. 检测服务

```csharp
using Cognex.VisionPro;
using Cognex.VisionPro.PMAlign;
using Cognex.VisionPro.Blob;
using Cognex.VisionPro.Caliper;
using Cognex.VisionPro.ID;

namespace DemoApp.Services;

public class PhoneFrameInspectionService : IDisposable
{
    private readonly CogPMAlignTool _pmAlign;
    private readonly CogFixtureTool _fixture;
    private readonly CogFindCircleTool _findCircle;
    private readonly CogBlobTool _blob;
    private readonly CogIDTool _idTool;
    private readonly CogToolBlock _toolBlock;

    public PhoneFrameInspectionService(string vppFile)
    {
        // 从 .vpp 文件加载工具链（推荐）
        if (!string.IsNullOrEmpty(vppFile))
        {
            _toolBlock = CogSerializer.LoadObjectFromFile(vppFile) as CogToolBlock;
            _pmAlign = _toolBlock.Tools["定位"] as CogPMAlignTool;
            _fixture = _toolBlock.Tools["坐标对齐"] as CogFixtureTool;
            _findCircle = _toolBlock.Tools["测孔径"] as CogFindCircleTool;
            _blob = _toolBlock.Tools["缺陷检测"] as CogBlobTool;
            _idTool = _toolBlock.Tools["读码"] as CogIDTool;
        }
        else
        {
            // 代码创建工具链（备选方案）
            _toolBlock = new CogToolBlock();
            _pmAlign = new CogPMAlignTool();
            _fixture = new CogFixtureTool();
            _findCircle = new CogFindCircleTool();
            _blob = new CogBlobTool();
            _idTool = new CogIDTool();

            _toolBlock.Tools.Add(_pmAlign, "定位");
            _toolBlock.Tools.Add(_fixture, "坐标对齐");
            _toolBlock.Tools.Add(_findCircle, "测孔径");
            _toolBlock.Tools.Add(_blob, "缺陷检测");
            _toolBlock.Tools.Add(_idTool, "读码");

            _toolBlock.Inputs.Add("InputImage", typeof(CogImage8Grey));
            _toolBlock.Outputs.Add("PassFail", typeof(bool));
            _toolBlock.Outputs.Add("ResultData", typeof(InspectionResult));
        }
    }

    public InspectionResult Inspect(CogImage8Grey image)
    {
        var result = new InspectionResult();

        try
        {
            // 步骤 1：定位
            _pmAlign.InputImage = image;
            _pmAlign.Run();

            if (_pmAlign.Results == null || _pmAlign.Results.Count == 0)
            {
                result.OverallPass = false;
                result.Message = "定位失败";
                return result;
            }

            result.LocateScore = _pmAlign.Results[0].Score;
            result.LocatePass = result.LocateScore >= 0.7;
            result.PositionX = _pmAlign.Results[0].GetPose().TranslationX;
            result.PositionY = _pmAlign.Results[0].GetPose().TranslationY;

            // 步骤 2：坐标对齐
            _fixture.InputImage = image;
            _fixture.Fixture = _pmAlign.Results[0].GetPose();
            _fixture.Run();

            CogImage8Grey alignedImage = _fixture.OutputImage as CogImage8Grey;

            // 步骤 3：测量孔径
            _findCircle.InputImage = alignedImage;
            _findCircle.Run();

            if (_findCircle.Results != null)
            {
                // 直径 = 半径 * 2
                result.HoleDiameter = _findCircle.Results.CircleRadius * 2;
                // 假设标准孔径为 8.0mm，公差 ±0.1mm
                result.HolePass = result.HoleDiameter >= 7.9
                               && result.HoleDiameter <= 8.1;
            }
            else
            {
                result.HolePass = false;
                result.Message = "孔径测量失败";
            }

            // 步骤 4：表面缺陷检测
            _blob.InputImage = alignedImage;
            _blob.RunParams.SegmentationParams.Polarity = CogBlobPolarityConstants.DarkBlobs;
            _blob.RunParams.SegmentationParams.ThresholdMode = CogBlobThresholdModeConstants.Absolute;
            _blob.RunParams.SegmentationParams.SegmentationThresholds.Clear();
            _blob.RunParams.SegmentationParams.SegmentationThresholds.Add(
                new CogBlobSegmentationThreshold(0, 80)
            );
            _blob.Run();

            result.DefectCount = 0;
            if (_blob.Results != null)
            {
                foreach (CogBlobResult r in _blob.Results.GetBlobs())
                {
                    if (r.Area >= 30) // 面积超过 30 像素视为缺陷
                    {
                        result.DefectCount++;
                    }
                }
            }
            result.DefectPass = result.DefectCount == 0;

            // 步骤 5：读 DPM 码
            _idTool.InputImage = alignedImage;
            _idTool.RunParams.CodeTypes = CogIDCodeTypeConstants.DataMatrix;
            _idTool.RunParams.DPM = true;
            _idTool.Run();

            if (_idTool.Results != null && _idTool.Results.Count > 0)
            {
                result.SerialNumber = _idTool.Results[0].DecodedData;
                result.ReadPass = !string.IsNullOrEmpty(result.SerialNumber);
            }
            else
            {
                result.ReadPass = false;
            }

            // 汇总判定
            result.OverallPass = result.LocatePass
                              && result.HolePass
                              && result.DefectPass
                              && result.ReadPass;

            result.Message = result.OverallPass ? "OK" : "NG";
        }
        catch (Exception ex)
        {
            result.OverallPass = false;
            result.Message = $"异常：{ex.Message}";
        }

        return result;
    }

    public void Dispose()
    {
        _toolBlock?.Dispose();
    }
}
```

### 2. 检测结果模型

```csharp
namespace DemoApp.Services;

public class InspectionResult
{
    // 总体结果
    public bool OverallPass { get; set; }
    public string Message { get; set; } = string.Empty;

    // 定位结果
    public bool LocatePass { get; set; }
    public double LocateScore { get; set; }
    public double PositionX { get; set; }
    public double PositionY { get; set; }

    // 孔径测量
    public bool HolePass { get; set; }
    public double HoleDiameter { get; set; }

    // 缺陷检测
    public bool DefectPass { get; set; }
    public int DefectCount { get; set; }

    // 读码结果
    public bool ReadPass { get; set; }
    public string SerialNumber { get; set; } = string.Empty;

    // 检测时间
    public DateTime InspectTime { get; set; } = DateTime.Now;
}
```

### 3. ViewModel

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using Cognex.VisionPro;
using DemoApp.Services;

namespace DemoApp.ViewModels;

public partial class InspectionViewModel : ObservableObject
{
    private readonly PhoneFrameInspectionService _inspectionService;
    private readonly CogAcqFifoTool _acqTool;

    [ObservableProperty]
    private CogImage8Grey? currentImage;

    [ObservableProperty]
    private string statusMessage = "就绪";

    [ObservableProperty]
    private InspectionResult? lastResult;

    [ObservableProperty]
    private int totalCount;

    [ObservableProperty]
    private int passCount;

    [ObservableProperty]
    private int failCount;

    [ObservableProperty]
    private bool isRunning;

    public InspectionViewModel(PhoneFrameInspectionService inspectionService)
    {
        _inspectionService = inspectionService;

        // 初始化采集工具
        _acqTool = new CogAcqFifoTool();
        _acqTool.Operator = new CogAcqFifoOperator();
        _acqTool.Operator.Init("Camera1");
        _acqTool.Operator.Trigger.TriggerModel = CogAcqTriggerModelConstants.Software;
    }

    [RelayCommand]
    private async Task RunInspectionAsync()
    {
        if (IsRunning) return;

        try
        {
            IsRunning = true;
            StatusMessage = "采集中...";

            _acqTool.Run();
            CurrentImage = _acqTool.OutputImage as CogImage8Grey;

            if (CurrentImage == null)
            {
                StatusMessage = "采集失败";
                return;
            }

            StatusMessage = "检测中...";

            var result = await Task.Run(() => _inspectionService.Inspect(CurrentImage));

            LastResult = result;
            TotalCount++;

            if (result.OverallPass)
            {
                PassCount++;
                StatusMessage = $"OK - {result.Message}";
            }
            else
            {
                FailCount++;
                StatusMessage = $"NG - {result.Message}";
            }
        }
        catch (Exception ex)
        {
            StatusMessage = $"异常：{ex.Message}";
        }
        finally
        {
            IsRunning = false;
        }
    }

    [RelayCommand]
    private void SaveCurrentImage()
    {
        if (CurrentImage != null)
        {
            string fileName = $"Frame_{DateTime.Now:yyyyMMdd_HHmmss}.bmp";
            CurrentImage.Save(fileName);
            StatusMessage = $"图像已保存：{fileName}";
        }
    }
}
```

## 三、运行效果

当系统运行时：

1. 点击"执行检测"，相机采集图像；
2. PMAlign 定位中框，显示定位十字线和分数；
3. FixturTool 摆正图像；
4. FindCircle、Blob、IDTool 在摆正图像上执行；
5. 结果显示在右侧面板，图像上叠加检测标记；
6. 统计信息实时更新。

## 总结

1. 完整的检测流程 = 采集 → 定位 → 坐标对齐 → 多项检测 → 结果判定；
2. 使用 CogToolBlock 统一管理工具链，使用 .vpp 文件实现配置与代码分离；
3. 每个检测项对应一个工具，每个工具输出独立的结果再汇总判定；
4. WPF 前端通过 CogDisplay 显示图像和检测结果，通过 CommunityToolkit.Mvvm 管理界面状态；
5. 这个案例可以轻松扩展为多型号支持、NG 图像保存、检测日志等功能。
