### 5.3 二维数组

#### 5.3.1二维数组定义方式

二维数组4种定义方式

1. 数据类型 数组名[行数] [列数];
2. 数据类型 数组名[行数] [列数] = {{数据1,数据2}{数据3,数据4}};
3. 数据类型 数组名[行数] [列数] = {数据1,数据2,数据3,数据4};
4. 数据类型 数组名[] [列数] = {数据1,数据2,数据3,数据4};

``````c++
int main(){
    //方式1
    //数据类型 数组名[行数][列数]；
    int arr[2][3];
    int[0][0] = 1;
    int[0][1] = 2;
    int[0][2] = 3;
    int[1][0] = 4;
    int[1][1] = 5;
    int[1][2] = 6;
    
    for(int i = 0; i<2;i++){
        for(int j=0;j<3;j++){
            cout<<arr[i][j]<<"  ";
        }
        cout<<endl;
    }
    
    //方式2
    //数据类型 数组名[行数][列数] = {{数据1,数据2},{数据3,数据4}};
    int arr2[2][3] = {
		{1,2,3},
		{4,5,6}
	};
    
    //方式3
    //数据类型 数组名[行数][列数] = {数据1,数据2,数据3,数据4};
    int arr3[2][3] = { 1,2,3,4,5,6 };
    
    //方式4
    //数据类型 数组名[][列数] = {数据1,数据2,数据3,数据4};
    int arr4[][3] = { 1,2,3,4,5,6 };
}
``````
